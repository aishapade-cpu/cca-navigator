# CCA Navigator — Roadmap

## Shipped
- Live trial search from ClinicalTrials.gov
- Plain-language summaries and fit scores via Claude
- Personalized doctor questions
- Researcher contact info + on-demand outreach message generator
- Save trials to revisit (localStorage)
- Landing page with About section
- Form data persistence on refresh

## Up next
- [ ] **Email alerts** — notify users when new CCA trials open
- [ ] **Location filtering** — filter to US-only or within X miles of a zip code

## Planned
- [ ] **Share a trial** — one-click copy of a trial summary to send to a doctor or family member
- [ ] **Biomarker/mutation filter** — filter trials by IDH1, FGFR2, HER2, and other common CCA mutations
- [ ] **Trial status tracking** — alert saved trials when recruiting status changes
- [ ] **Doctor-facing summary** — exportable one-page PDF of saved trials for appointments

## Ideas (not yet prioritized)
- Expand to other rare cancers (pancreatic, gallbladder)
- User accounts for cross-device saved trials
- Partnership with Cholangiocarcinoma Foundation
